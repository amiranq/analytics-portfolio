# Анализ данных в недвижимости

## Описание проекта

Данный проект посвящен анализу данных о сделках с жилой недвижимостью и построению регрессионных моделей для выявления факторов, оказывающих наибольшее влияние на стоимость жилья. Работа включает разведочный анализ данных (EDA), построение базовой модели множественной регрессии и ее последовательное улучшение на основе выявленных закономерностей.

Цель проекта — **выявить ключевые признаки, влияющие на цену объектов недвижимости, и построить линейную регрессионную модель с хорошим качеством прогнозирования**.

---

## Используемые данные

В проекте используется датасет со следующими основными признаками:
- дата сделки;
- возраст дома;
- расстояние до ближайшей станции метро (MRT);
- количество магазинов поблизости;
- географические координаты объекта (широта и долгота);
- цена жилья за единицу площади (целевая переменная).

Все объекты расположены в пределах одного района в г. Новый Тайбэй, Тайвань.

---

## Структура работы

Проект включает следующие основные этапы:

1. **Разведочный анализ данных (EDA)**  

2. **Построение базовой модели**  

3. **Статистический анализ и добавление новых признаков**  

4. **Построение улучшенной модели и сравнение результатов**  

---

## Результаты

Финальная модель с логарифмированным расстоянием до метро и учетом пространственных кластеров показала заметное улчшение качества:

$R^2(train): 0,554 → 0,640$

$R^2(test): 0,581 → 0,639$

$RMSE(train): 9,323 → 8,379$

$RMSE(test): 7,950 → 7,377$

Ключевые факторы, влияющие на стоимость жилья:
- расстояние до ближайшей станции метро;
- пространственное положение объекта;
- возраст дома;
- количество магазинов поблизости.
