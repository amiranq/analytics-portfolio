# Стратегия повышения прибыли DNS

Представлены 3 ключевые презентации нашей команды из курса "Бизнес-студия" в Центральном университете.

## Цель
Повысить прибыль DNS за счет через улучшение ключевых точек контакта (онлайн/офлайн/логистика/сервис).

## Что исследовали
- **Онлайн:** дизайн/UX, каталог/фильтры/поиск, скорость, поддержка, актуальность наличия  
- **Офлайн:** планировка, простаивающие зоны, выкладка/экспозиция, персонал  
- **Операции:** логистика и выдача, сервисный центр, стандарты сервиса сотрудников

## Что сделали
1) **Field study и анализ сайта**: наблюдения в магазинах, разбор сайта по CJM, фиксация проблем логистики и СЦ  
2) **Выводы**: позитивные/негативные, причины и точки потерь в пути клиента  
3) **Идеи → инициативы**: брейншторм ~50 идей, кластеризация в инициативы  
4) **Приоритизация**: оценка по **RICE**, выбор топ-инициативы  
5) **Проработка топа**: редизайн сайта - MVP и дизайн A/B-теста, план внедрения и риски

## 5 приоритетных областей
1) **Дизайн/UX сайта**  
2) **Логистика**  
3) **Площади и планировка офлайна**  
4) **Сервисный центр**  
5) **Сотрудники/качество сервиса**

## Ключевая инициатива
**Редизайн сайта DNS** (упрощение дизайна и снижение когнитивной нагрузки).

### 3 кластера проблем → решения (MVP)
- **Инфоперегруз** → меньше вкладок/баннеров, убрать дубли, четкие CTA  
- **Визуальная перегрузка** → крупнее шрифт, контраст, акценты на цене/CTA  
- **Навигация/поиск** → структура каталога 3 уровня, умные фильтры/подсказки, линейный checkout

## MVP + A/B тест
- **Метрики:** CR, bounce rate, add-to-cart, completion checkout, AOV  
- **Группы:** A (старый) vs B (MVP), рандомизация  
- **Порог эффекта (MDE):** фиксируем заранее  
- **Эксперимент:** идет до набора выборки для стат. значимости → решение масштабировать/доработать/откатить

## План внедрения
UX-аудит → прототипы → модульная разработка → A/B → масштабирование → дизайн-система и постоянные итерации.

## Фреймворки и методы
- **Double Diamond**, **Field Study**, **CJM**
- **RICE**
- **MVP**, **A/B testing**
- Бенчмаркинг конкурентов
- Brainstorming, кластеризация
